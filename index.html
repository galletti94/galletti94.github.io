Projects

Projects I've done over the years. You will find the content by clicking on the icons.

  • ATS-ML

    Currently, I am working with Professor Hongwei Xi at Boston University to unleash the Machine Learning potential of the functional programming language ATS by providing support for co-programming with R.

  • FRP Practice

    I am building a small collection of buttons and counters in a Function Reactive Programming style to get practice with Bacon.js

  • Photoshare

    Photoshare is a web-based photo-sharing social application that I developped from scratch to get familiar with python flask and jinja2. IMPORTANT NOTE: For security reasons please DO NOT log in with a real email or a password you typically use.

  • Mentor Matching

    In collaboration with Professor Evimaria Terzi and BU Spark! at Boston University, I implemented matching algorithms to pair innovative students to experienced mentors.

  • Kaggle Competitions

    I regularly participate in Kaggle Competitions. It pushes me to learn new tools and to improve my programming (both in productivity and in quality).

  • Network of Queues Simulations

    Using simpy in python I created a simulation of various queue models. If you are interested in learning concurrent programming in Python or need a base from which to start your own simulation, this could be a good place to start.

  • Long-Range Dependence

    This is the final paper I wrote for a graduate probability class I took at Boston University. If you are interested in learning about time series and brownian motion, and stochastic processes derived from them, this could be for you. The presentation is here

  • Probability of two integers being Coprime

    After reading Apostol's book Introduction to Analytic Number Theory, I wrote this small paper summarizing a fun instance of number theory intersecting with probability.
