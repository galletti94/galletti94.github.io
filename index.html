Projects

Projects I've done over the years.

  • ATS-ML

    Worked with Professor Hongwei Xi at Boston University to combine functional programming with Machine Learning by providing support for co-programming between ATS and R. Also experimented with some deep learning.

  • Tweet Analysis

    I collected and analyzed tweets using the twitter API, tweepy, and mongodb. Check out the jupyter notebook for a walk through of my process!

  • Functional Blockchain

    Blockchain and smart contract language with CLI. All implemented in ATS, from scratch and in a functional programming style.

  • Functional Programming

    A collection of algorithms I implemented in ATS. This functional style of programming is highly productive and when applied to computation on large data, it is often superior as it is extremely memory efficient. Please check out the following GIFs!

  • blur for Android

    blur is an android app that uses your camera to recognize digits and letters, and learns from experience by storing photos in a database

  • FRP

    A fun collection of buttons and counters in a Functional Reactive Programming (FRP) style using Bacon.js

  • Team Management

    I built a platform for consultants to enter their weekly hours and see various visualizations of their teams data. The director of BU Spark! is provided an overview of all the projects and their statuses. This platform is now mandatory for most consulting projects at BU Spark!

  • Photoshare

    Photoshare is a web-based photo-sharing social application that I developped from scratch to get familiar with python flask and jinja2. IMPORTANT NOTE: For security reasons please DO NOT log in with a real email or a password you typically use.

  • Mentor Matching

    In collaboration with Professor Evimaria Terzi and BU Spark! at Boston University, I implemented matching algorithms to pair innovative students to experienced mentors.

  • Docker Images

    Docker is awesome! I am slowly but surely adding Dockerfiles to all my github repositories to make it easier and faster for people to clone, use, and test my code. Check out some of my docker images and avoid having a cluttered computer!

  • Network of Queues

    Using simpy in python I created a simulation of various queue models. If you are interested in learning concurrent programming in Python or need a base from which to start your own simulation, this could be a good place to start.

  • Kaggle Competitions

    I regularly participate in Kaggle Competitions. It pushes me to learn new tools and to improve my programming (both in productivity and in quality).

  • Long-Range Dependence

    This is the final paper I wrote for a graduate probability class I took at Boston University. If you are interested in learning about time series and brownian motion, and stochastic processes derived from them, this could be for you. The presentation is here

  • Probability of Coprime-ness

    After reading Apostol's book Introduction to Analytic Number Theory, I wrote this small paper summarizing a fun instance of number theory intersecting with probability.
